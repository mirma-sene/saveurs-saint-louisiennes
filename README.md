# Saveurs Saint-Louisiennes

## Description
Ce projet consiste à concevoir et développer une landing page de présentation d’un restaurant.

## Objectifs
- Créer une maquette avec Figma
- Organiser le travail avec Trello
- Développer le site avec HTML, CSS, Bootstrap et JavaScript

## Sections du site
- Navbar
- Header
- Menu des plats
- Galerie
- Témoignages
- Réservation
- Contact
- Footer

## Outils utilisés
- Figma
- Trello
- Git & GitHub
